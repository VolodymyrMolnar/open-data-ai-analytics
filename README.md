# Powerlifting Data Analytics - Version A

## Мета проєкту
Цей проєкт присвячений дослідженню відкритих даних про результати змагань з пауерліфтингу. Основна ціль — вивчити зв'язок між антропометричними даними атлетів (вага, стать, вік) та їхніми силовими показниками, використовуючи методи аналізу даних та машинного навчання.

## Джерело даних
Дані взяті з відкритого репозиторію **OpenPowerlifting**:
* [OpenPowerlifting Database](https://www.kaggle.com/datasets/open-powerlifting/powerlifting-database)
* Набір містить інформацію про сотні тисяч виступів атлетів, включаючи їхню вагу, вік, стать, результати у кожній вправі (Squat, Bench, Deadlift) та використане екіпірування.

## Питання та гіпотези для аналізу
1.  **Кореляція маси та сили:** Чи існує лінійна залежність між власною вагою атлета та його результатом у присіданнях ($Best3SquatKg$)? Гіпотеза: зі збільшенням вагової категорії абсолютна сила зростає, але темпи приросту сповільнюються у важких вагових категоріях.
2.  **Вікові зміни:** Як змінюється середній результат у жимі лежачи залежно від вікової категорії (наприклад, порівняння груп 20–25 років та 40–45 років)? Гіпотеза: пік сили в жимі припадає на вік 30–35 років.
3.  **Відносна сила за статтю:** Хто має вищий середній показник відносної сили за формулою $Wilks$ або $IPF\ Points$ — чоловіки чи жінки? Гіпотеза: жінки мають конкурентні або навіть вищі показники відносної сили у легких вагових категоріях.

## Структура репозиторію
* `data/` — папка для збереження датасету (ігнорується Git).
* `notebooks/` — Jupyter-ноутбуки з розвідувальним аналізом (EDA).
* `src/` — скрипти для завантаження та обробки даних.
* `reports/figures/` — графіки та візуалізації результатів.
